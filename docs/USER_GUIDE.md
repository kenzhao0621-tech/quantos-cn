## 1. 安装与登录

```bash
git clone https://github.com/kenzhao0621-tech/netlify-demo.git
cd netlify-demo
make bootstrap
cp .env.example .env   # 填入 TUSHARE_TOKEN（推荐）
make app
```

浏览器打开：**http://127.0.0.1:8787/portal**

1. 身份选 **「新手投资者（推荐）」**
2. 阅读并勾选 **风险与法律确认**
3. 从 **「新手入门」** 页按四步操作

### 常见问题

| 现象 | 处理 |
|------|------|
| 券商按钮报 403 / 权限不足 | 用 **新手投资者** 登录；点「备用官方链接」或下载券商 App；勿依赖自动化弹窗 |
| 接口 404 | 重启门户：`bash scripts/start-portal.sh` |
| 选股无结果 | 新手入门 → **① 更新数据**；检查 Tushare Token |
| 选股总是超时 | 改用 **收盘数据** 模式；勿默认「实时」；先更新日线数据 |
| 同花顺无法网页登录 | 正常：请下载同花顺 App → 绑定券商账户 → 在 App 内下单 |

## 3. 推荐四步流程

### ① 更新数据

新手入门 → **更新数据**（或高级·数据 → 市场同步）

确认顶栏 **「数据」** 显示 `EOD_OK` 或 `INTRADAY_OK`。

### ② 智能选股

1. 打开 **智能选股**
2. 模式选 **收盘数据（推荐·快速）** — 约 1 秒出结果
3. 总资金填 **5000**（或你的实际预算）
4. 可选：设置 **最低/最高股价**；勾选「按资金自动限制最高价」
5. 点 **运行选股**，查看推荐列表、**增强版选股指南**与一手数量

> **实时智能** 需先在「高级·数据」刷新实时行情；否则会超时或退回收盘因子。

### ③ 模拟练习

1. 打开 **模拟练习** → **启动模拟**
2. 观察 3–5 个交易日，理解 T+1（今日买明日才能卖）
3. 模拟 **不产生真实盈亏**

### ④ 券商助手（真实交易须本人确认）

1. **券商助手** → 选择券商（默认东方财富）
2. 点 **连接并打开登录页** → 在**你自己的浏览器**（Chrome/Safari）完成登录
3. 若页面显示 **403**：点 **备用官方链接**，或下载券商 App 登录（部分券商会拦截自动化窗口）
4. 回到智能选股，对心仪标的点 **实盘辅助**
5. **在券商官方 App 亲自点击确认** — 系统只预填，不自动扣款

支持的辅助路径包括：浏览器打开官方交易页、导出 CSV、本地 QMT 目录落单（进阶）。**不保存** 交易密码与短信验证码。

## 4. 每日算法学习

每次选股会写入学习账本。系统对照后续走势统计 forward return，用于 **改进算法**，并在「新手入门」页展示摘要。

- 这是 **研究与校准**，不等于未来盈利保证
- 模型可能失效，请始终以模拟盘为先

---

## 5. 免责说明（必读）

1. **不构成投资建议**：选股结果、回测、收益率区间均为模型输出，仅供参考。
2. **不自动真实下单**：可打开券商页面并预填订单，最终成交须你在券商官方渠道确认。
3. **不保存密码**：不收集交易密码、资金密码、短信验证码。
4. **A 股交易规则**：T+1、100 股整数倍、涨跌停与停牌可能导致无法成交。
5. **模型风险**：历史表现不代表未来；样本外验收可能未通过，部署状态以系统标注为准（当前多为 `SHADOW_ELIGIBLE` / 模拟优先）。
6. **数据延迟**：行情与财务数据可能存在延迟或缺失，请以券商终端为准。

完整法律措辞见门户登录弹窗及 `docs/COMPARISON.md`。

## 6. 风险与紧急停机

- **亏损额度**：在偏好中设置可承受的最大回撤比例
- **Kill Switch（紧急停机）**：阻断新信号；需按流程申请解除
- 真实资金通道默认关闭；即使开启也须 **人工在券商端确认**
