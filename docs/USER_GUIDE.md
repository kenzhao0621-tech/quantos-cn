## 1. 安装与登录

完整安装见 **[INSTALL.md](INSTALL.md)**（含 Windows）。

```bash
git clone https://github.com/kenzhao0621-tech/netlify-demo.git quantos-cn
cd quantos-cn
make bootstrap && cp .env.example .env
make app
```

浏览器：**http://127.0.0.1:8787/portal**

1. 点击 **进入平台**
2. 阅读并确认 **风险与法律**
3. 从 **智能选股** 开始（默认首页）

### 常见问题

| 现象 | 处理 |
|------|------|
| 端口 8787 占用 | `make portal-stop`（Windows：`scripts\stop-portal.ps1`） |
| 接口 404 | 重启：`bash scripts/start-portal.sh` |
| 选股无结果 | 高级·数据 → 市场同步；检查 Tushare Token |
| 实时选股较慢 | 首次约 20–90 秒；2 分钟内重试走缓存；休市可用收盘模式 |
| 券商 403 | 使用「备用官方链接」或券商 App |

## 3. 推荐流程

### ① 智能选股（收盘模式 · 最快）

1. 打开 **智能选股**
2. 模式：**收盘数据（推荐·快速）** — 约 1 秒
3. 总资金：**5000**（或你的预算）
4. **运行选股**
5. **点击任意股票行** → 弹窗查看得分、因子、智能体评语
6. 可 **导出 PDF** 或 **加入 Paper**

### ② 实时智能选股

1. 模式选 **实时智能**
2. 点 **运行实时智能选股**（内置行情刷新，无需先切页）
3. 首次可能需 20–90 秒；顶栏 **实时** pill 显示就绪状态

### ③ 策略验证

1. 打开 **策略验证**
2. **验证昨日选股** — 对照 T+1 实际走势
3. **策略自验证学习** — 获取参数调整建议（研究用途）

### ④ 模拟练习

1. **模拟练习** → **启动模拟**
2. 从选股结果 **加入 Paper 模拟组合**
3. 观察 3–5 个交易日，理解 T+1
4. 模拟 **不产生真实盈亏**

### ⑤ 券商助手（真实交易须本人确认）

1. 选择券商 → **连接并打开登录页**
2. 在 **你自己的浏览器** 完成登录
3. 回到选股，对标的点 **实盘辅助**
4. **在券商 App 亲自确认** — 系统只预填，不自动扣款

## 4. 每日算法学习

选股结果写入学习账本，系统对照后续走势统计 forward return，在 **策略验证** 页展示摘要。

- 这是 **研究与校准**，不等于未来盈利保证
- 模型可能失效，请始终以模拟盘为先

---

## 5. 免责说明（必读）

1. **不构成投资建议**：选股、回测、收益率区间均为模型输出。
2. **不自动真实下单**：可预填订单，成交须你在券商官方渠道确认。
3. **不保存密码**：不收集交易密码、短信验证码。
4. **A 股规则**：T+1、100 股整数倍、涨跌停与停牌可能导致无法成交。
5. **模型风险**：历史表现不代表未来。
6. **数据延迟**：请以券商终端为准。

完整法律措辞见门户登录弹窗及 [COMPARISON.md](COMPARISON.md)。
