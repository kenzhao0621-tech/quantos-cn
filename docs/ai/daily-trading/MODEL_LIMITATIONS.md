# Model Limitations

- Sector/stock pipeline uses deterministic fixtures for validation; live universe screening not fully wired to AKShare.
- Scores are transparent heuristics, not ML alpha guarantees.
- Regime classification is simplified (index change + breadth).
- Cannot detect all rumor sources; official catalysts preferred.
- Heuristic web-content safety is not perfect against novel injection.
- Backtest engine skill is scaffold only.
