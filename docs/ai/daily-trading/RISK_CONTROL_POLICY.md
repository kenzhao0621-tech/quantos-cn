# Risk Control Policy

1. **Capital preservation** over trade frequency.
2. **Single-name cap**: 10–20% of trading capital (simulation).
3. **Stop-loss required** for every primary candidate; no stop → no entry.
4. **Minimum reward:risk** 1.5:1 for primary list.
5. **ST / suspended / limit-up** excluded from primary recommendations by default.
6. **Stale data** → no live intraday entry recommendations.
7. **Strong bear** regime → default `NO TRADE`.
8. User confirms all real orders in brokerage UI.
