# UI and Visual Pipeline

## Order (spec §5)

1. Requirements → 2. IA → 3. User flows → 4. Design system → 5–7. Typography/color/spacing  
8. Wireframes (`diagram-architect`) → 9. Figma MCP (when configured) → 10. `frontend-design`  
11. `webapp-testing` → 12. Screenshots → 13. Screenshot QA (UNRESOLVED upstream)  
14. Accessibility (manual + Playwright a11y) → 15. `impeccable` polish → 16. Re-test

## Skill boundaries

| Skill | Role |
|-------|------|
| ui-ux-pro-max | Design research only — not primary coder |
| frontend-design | Implementation |
| impeccable | Final polish — no core logic rewrites |
| webapp-testing | Browser automation primary |
